# Discuz! X 主题列表信息流插件

## 应用介绍

您还在纠结主题列表无法自动向下滚动么？还在羡慕那些走上康庄大道的信息流产品么？

本插件特别为此设计，让用户只要鼠标下拉就可以不停看到最新主题。

只要您内容够多，带宽够大，就永远不会看到下一页！

**更多特色功能欢迎您安装体验~**

## 注意事项

1. 本插件仅支持 PC 端，暂不支持移动端。

2. 本插件不支持低版本 IE 浏览器（最低 IE 9 ），更低的版本效果等同于未安装本插件。
